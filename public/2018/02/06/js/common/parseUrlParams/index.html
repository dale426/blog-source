知识点 来自W3C

编码encodeURI() 解码decodeURI()
  1. 把字符串作为 URI 进行编码。
  2. 对 / ? : @ & = + $ # 不会转义。
编码 encodeURIComponent() 解码 decodeURICompnent()
  1. 把字符串作为 URI 组件进行编码。
  2. 不会对 ASCII 字母和数字进行编码，
  3. 也不会对这些 ASCII 标点符号 - _ . ! ~ * ‘ ( ) 进行编码：
  4. 其他字符（比如 ：;/?:@&=+$,# 这些用于分隔 URI 组件的标点符号），都是由一个或多个十六进制的转义序列替换的。
